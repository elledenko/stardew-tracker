# Stardew Tracker — Research Brief

## Market Overview
- 35M+ copies sold across all platforms
- r/StardewValley: 2M+ subscribers
- Consistently 50-100k concurrent Steam players
- Strong casual/cozy gaming community with high engagement

## Competitor Analysis

| Tool | What It Does | Strengths | Gaps |
|------|-------------|-----------|------|
| Stardew Checkup | Upload save file, see completion % | Great perfection tracking | Read-only, no planning |
| Stardew Valley Wiki | Comprehensive reference | Exhaustive data | No personalization |
| Stardew Planner | Visual farm layout designer | Drag-and-drop | No crop scheduling |
| Stardew Profits | Crop profit calculator | Good math | Single-purpose |
| Companion apps | Gift guides, calendar | Portable | Static lookups only |

## Gap Analysis
No existing tool provides:
- Per-in-game-day logging
- Cross-system tracking (gifts + crops + activities in one place)
- Progress visualization over time
- Daily planner/journal functionality

## Target User Profile
- Stardew Valley players (all platforms)
- Players who want to remember what they did
- Completionists tracking gifts and relationships
- Min-maxers optimizing daily gold

## Go/No-Go: GO
Large, engaged community with zero daily companion tools. Clear gap in the market.
